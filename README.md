Admeen
======

>就如同很多人前光鲜的妹子回家后都不爱洗内衣一样，那些前端漂亮的web产品通常都有一个奇丑无比老气横秋的后台。Admeen，旨在减少后台前端的开发成本，提高用户体验，为admin的生命增加长度，还有宽度：）

优点
----------------------
+ 基于bootstrap，无学习成本
+ 支持移动设备&webAPP
+ 包含人世间的后台能用到的几乎所有功能
+ 可扩展，包含数种功能的php demo
+ 绝佳的体验
